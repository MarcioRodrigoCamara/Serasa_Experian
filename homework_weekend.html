Historia da internet

O que é internet?

A Internet é uma rede global de computadores interconectados que usam o protocolo de comunicação TCP/IP para compartilhar informações e recursos. Ela permite que as pessoas acessem e compartilhem informações, comuniquem-se uns com os outros e usem aplicativos online de qualquer lugar do mundo. A Internet foi criada na década de 1960 e tem evoluído rapidamente desde então, tornando-se uma parte importante da vida cotidiana e da economia global.

O que foi a ARPANET?

A ARPANET (Advanced Research Projects Agency Network) foi a primeira rede de computadores de grande escala a ser criada. Ela foi projetada e desenvolvida pela agência de pesquisa do Departamento de Defesa dos Estados Unidos (DARPA) nos anos 60. A ARPANET foi o precursor da Internet atual, e foi criada com o objetivo de permitir que as universidades e instituições de pesquisa compartilhem recursos de computação e informações. Ela usa o conceito de endereços de rede (IP) e o protocolo de comunicação TCP/IP, que ainda são usados ​​na Internet atual. A ARPANET foi desativada em 1990, mas muitos de seus princípios e tecnologias continuam a ser usados ​​hoje.

Como surgiu a internet no Brasil?

A internet no Brasil surgiu na década de 1990, com o aumento do uso de computadores pessoais e a popularização da tecnologia. O governo brasileiro, através de sua estatal Telebras, criou o Projeto Internet Brasileira (IB), que tinha como objetivo levar a internet a universidades e instituições de pesquisa. A primeira conexão de internet comercial foi estabelecida em 1991 pela empresa Prestel, que oferecia serviços de correio eletrônico e acesso às bases de dados. A partir daí, outras empresas passaram a oferecer serviços de internet, e o número de usuários cresceu rapidamente. Atualmente, a internet é amplamente utilizada no Brasil para comunicação, acesso à informação, entretenimento e negócios.

Historia da WEB

O que é a WEB?

A World Wide Web, ou simplesmente, a Web, é um conjunto de documentos e outros recursos interligados, como imagens e vídeos, que são acessíveis através da Internet. A Web foi criada por Tim Berners-Lee em 1989, enquanto ele trabalhava no CERN (Organização Europeia para a Pesquisa Nuclear). A Web usa um sistema de hipertexto, o que permite que os usuários naveguem entre documentos (ou "páginas") através de links (ou "hiperlinks"). Isso permite que as pessoas se encontrem, acessem e compartilhem informações facilmente, independentemente de onde elas estejam no mundo. A Web é a principal maneira pela qual as pessoas acessam a Internet hoje, e é usada para uma variedade de propósitos, incluindo comunicação, compras, entretenimento, educação e negócio

Como e quando surgiu a WEB?

A World Wide Web (WWW ou Web) surgiu em 1989, quando o cientista do computador britânico Tim Berners-Lee, trabalhando no CERN (Organização Européia para a Pesquisa Nuclear), percebeu a criação de um sistema de hipertexto que permitiria aos usuários acessar informações de forma fácil e interativa. Ele desenvolveu as primeiras versões dos protocolos HTTP (Hypertext Transfer Protocol), HTML (Hypertext Markup Language) e URL (Uniform Resource Locator), que são os pilares da Web até hoje. Em 1991, Berners-Lee lançou a primeira versão pública da Web, tornando-a acessível a qualquer um com uma conexão à internet. A partir daí, a Web cresceu rapidamente e se tornou uma parte vital da vida cotidiana, oferecendo uma variedade de recursos e serviços, como e-mails, compras on-line, mídias sociais e muito mais

Como funciona a WEB?

A World Wide Web (Web) funciona através de um conjunto de protocolos e tecnologias que permitem acessar, compartilhar e trocar informações na internet. Os principais protocolos envolvidos no funcionamento da Web são o HTTP (Hypertext Transfer Protocol) e o HTML (Hypertext Markup Language).

O HTTP é um protocolo de rede que permite a comunicação entre um navegador web (como o Chrome, Firefox, Safari) e um servidor web. Quando um usuário digita uma URL (Uniform Resource Locator) em seu navegador ou clica em um link, o navegador envia uma requisição HTTP para o servidor, solicitando acesso à página específica. O servidor, por sua vez, envia uma resposta HTTP que contém a página HTML solicitada. O navegador então interpreta e exibe o conteúdo da página HTML para o usuário.

O HTML é uma linguagem de marcação que permite estruturar e formatar o conteúdo de uma página web, criando elementos como títulos, parágrafos, imagens, links e formulários. O HTML também permite a incorporação de outras tecnologias, como o JavaScript e o CSS, que permitem a criação de páginas web mais interativas e com melhor design.

Outros protocolos e tecnologias também são usados ​​na funcionalidade da Web, como o DNS (Domain Name System), que convertem os endereços da web em endereços IP, e o SSL/TLS, que garantem a segurança das comunicações entre o navegador e o servidor.

Desenvolvimento WEB

O que é e como surgiu HTML?

HTML (Hypertext Markup Language) é uma linguagem de marcação utilizada para estruturar e formatar o conteúdo de documentos eletrônicos, especificamente para a criação de páginas web. Ele foi criado por Tim Berners-Lee, o inventor da World Wide Web, enquanto ele trabalhava no CERN (Organização Européia para a Pesquisa Nuclear) em 1989. Ele desenvolveu o HTML como uma maneira de estruturar o conteúdo de documentos eletrônicos de forma que poderia ser acessado através de hiperlinks.

A primeira versão do HTML, conhecida como HTML v0.9, tinha apenas algumas tags básicas, como 'title' , 'h1' , 'p' , 'a' e 'img' . Com o passar do tempo, novos recursos e tags foram adicionados, permitindo a criação de documentos cada vez mais complexos e interativos. Atualmente, existe uma versão HTML5, que inclui tags para elementos como vídeo, áudio, formulários e localização geográfica, entre outros.'

HTML é uma das principais tecnologias da Web e é usado em conjunto com outras tecnologias, como CSS e JavaScript, para criar páginas web com design e interatividade avançada.

O que é e como surgiu CSS?

CSS (Cascading Style Sheets) é uma linguagem de folhas de estilo utilizada para descrever a apresentação de documentos escritos em uma linguagem de marcação, como o HTML. Ele foi criado por Håkon Wium Lie e Bert Bos em 1994, enquanto trabalhavam na CERN (Organização Européia para a Pesquisa Nuclear). Eles desenvolveram o CSS como uma maneira de separar a estrutura do conteúdo de uma página (escrita em HTML) da sua apresentação visual (escrita em CSS).

Com CSS, os desenvolvedores web podem aplicar estilos (como cores, fontes, espaçamento e layout) a elementos HTML de forma consistente em toda uma página ou site. Isso permite que eles alterem a aparência de um site inteiro ao modificar apenas uma folha de estilo, em vez de editar cada página individual. Além disso, as folhas de estilo tornam mais fácil para os desenvolvedores manter o design consistente em diferentes dispositivos e tamanhos de tela.

Atualmente, existe uma versão CSS3, que inclui recursos para trabalhar com animações, transforma 3D, layouts flexíveis, entre outros. Além disso, também existe o CSS Grid Layout, que permite criar layouts bidimensionais avançados e flexíveis.
